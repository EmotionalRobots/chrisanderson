About this Project:

This is a small project made in React using Redux.

When a time slot is clicked, a modal appears which asks for a name and number. As a name or number is entered, the time slot selected will change to red and be marked as "Reserved". If a reserved time slot is clicked on again, the modal will pop up with the name and phone number for that appointment prepopulated. Users will be able to edit the name and phone number to change the user for the appointment.

CAPTCHA Selector

Just a small project I did for fun - a custom CAPTCHA selector which fetches random images, and displays them with css overlays styled after clicks. Each image has a boolean value associated to if it is, (or isn't), an image of a bird - and this is randomized at render time. Contained within an Angular Component

Weather Forecast

This project hits OpenWeatherAPI for individual days, as well as the 5 day forcast for a given zip code. The highs and lows for each day are then graphically represented.

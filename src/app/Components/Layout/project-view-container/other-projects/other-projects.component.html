CAPTCHA Selector

Just a small project I did for fun - a custom CAPTCHA selector which fetches random images, and displays them with css overlays styled after clicks. Images have a boolean value tied to if it is (or isn't) an image of a bird - and this is also randomized at render time. Contained within an Angular Component

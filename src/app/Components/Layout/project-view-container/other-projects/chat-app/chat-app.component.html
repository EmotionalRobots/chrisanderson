About this Project:

This component is part of a larger, full stack prototype application that I developed. The full web app includes user authentication, and once a user is logged in they are given access to the chat component shown below. The text boxes are created with bootstrap flex boxes, and using ng-class to dynamically style the message elements based on the logged in status. All messages are stored on the server, and the chatArea is automatically scrolled to the bottom whenever a new message is sent or received.
